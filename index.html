HTML

  • Hypertext Markup Language (HTML) is one of the three cornerstone languages used in web development: HTML, CSS, and JS.
  • HTML is used to create the structure of a webpage.
  • HTML is made up of elements and an element is a type of content on a webpage.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.
  • Two elements are especially important to understanding how the structure works: the head and the body.
  • The head element contains information, or metadata, that helps the browser to render the page correctly. Metadata is data about our webpage that the browser uses to effectively serve our webpage to users. Elements included within the head are not visible when a user looks at a webpage.
  • The body element defines the body of the document, which contains the content that is visible to the user. All the elements that we want the user to see and interact with must be placed inside this element.
  • The body element is an example of a semantic HTML element. It is important to use semantic HTML elements, because they make your HTML easier to read by other developers and the browser and more accessible to a variety of users (assistive technologies).
  • An HTML attribute defines additional information about an element.
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • Elements that are nested inside other elements are called child elements. The element that holds all the children is known as the parent element.
  • The Alt text is important for screen readers and other assistive technologies. Alt text can also improve search engine rankings for your website.
  • Section elements are semantic elements that define a section in a document that contains thematic content.

CSS

  • Cascading Style Sheets (CSS) is used to add style to a webpage, it is a programming language used by developers to define how webpage content is presented to users.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • The most common way to style a webpage is by using an external CSS style sheet. This is preferred in most cases because it allows developers to keep all their CSS rules in a separate file, which makes design changes easier (Separation of concerns).
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Every HTML element is essentially a box (CSS box model).
  • To apply a rule to all the elements visible on the page we use a special wild card symbol, or *.
  • We combined two selectors into a single CSS rule with a comma, allowing the declarations to affect both selectors.
  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained.
  • text-align is used as one way to center a title or heading on a webpage.
  • The img selector assigns property values to any image element in the markup.
  • To link an external CSS style sheet to an HTML file, you need to add the link to the style sheet in the head section of the HTML file.
  • A link element creates the connection to the external file. The rel attribute specifies the relationship between the current document and the linked document resource. The href attribute specifies the location (URL) of the external resource.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • The class attribute can be assigned to the element in the HTML, enabling us to freely choose which elements will be assigned the CSS rule.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: stages the new modification in the branch we are working in.
  • git checkout branch-name: changes the branch we are working on.
  • git commit -m “your message here”: commits the changes locally.
  • git pull origin main: checks if our local branch is up to date with the main branch in our remote repository, before pushing up our changes.
  • git push origin branch-name: pushes up the changes to the remote repository.

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites. Using JavaScript, you can power your websites with effects, interactivity, data communication, and much more.
  • Variables are containers that store values, to declare a variable, we use the var keyword.
  • To link an JS file to an HTML file, you need to add the link to JS file in the body section of the HTML file.
  • Developers use the console.log method daily to debug and test the code that they write.
  • Keyboard commands to open the console: Command+Option+I on macOS, Control+Shift+I on Windows.
  • Control flow is the order in which a computer executes code in a script. In JS, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
  • Conditional statements and loops are used to control the order in which code is executed, or if it is even executed at all.
  • If we use the if statement, code will only execute if the statement meets a condition, or is true. If it is true, then the computer will do whatever action we ask it to do in between the curly brackets. If the condition is not true or has a value that is not truthy, then the computer will ignore whatever is in between the curly brackets and move on.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way. Arrays are especially useful when we want to perform a task on multiple pieces of data at once.
  • A single array can store different types of data, such as strings, numbers, and so on.
  • The index is a unique number, which identifies the location of the data item in the array. It is used to access individual data inside the array.
  • Arrays are zero-indexed and sequential. This means that the first item in any array has an index of 0, not 1. They are also whole numbers that increase by 1 each time.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • Commenting out code makes a part of the code inactive without deleting it, just in case you need it later. To comment out put /* before and */ after the block of code.
  • A function is a set of instructions that tells the computer how to perform a certain task.
  • You must call a function by name in order to execute it.
  • To create a function, we use the function keyword. Then we give our function a unique name. We will use this unique name whenever we call the function.
  • A function's name is followed by parentheses ().
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number.
